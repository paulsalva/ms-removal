# MS Removal Static Site

Static HTML/CSS site built for Vercel deployment.

## Pages
- `index.html` - main landing page
- `construction-debris.html` - Construction Debris Removal / Concrete Disposal
- `tree-brush-removal.html` - Tree & Brush Removal / Landscape Waste / Material Delivery
- `property-cleanouts.html` - Property Cleanouts / Landlord Turnover Cleanup

## Before Launch
Replace placeholders:
- Phone links: `tel:+10000000000`
- Visible phone copy: `Call for a Quote`, or add the real number
- Form action: currently `mailto:info@example.com`
- Add actual service area language once confirmed
- Add real reviews, job photos, and proof points if available

## Vercel
Deploy as a static site from the repo root. No build command needed.
